# Expanso Design System - Local Reference

This file contains the official Expanso design system for use in all presentations and materials.

> **Source**: Official Expanso Color Palette from design-system repository  
> **Last Updated**: 2025-10-19

## Color Palette at a Glance

### Primary Actions & Branding

```
Expanso Primary:    violet-700  #6823CD  (CTA buttons, primary actions)
Expanso Hover:      violet-800  #501E99
Bacalhau Primary:   blue-600    #0055FF  (Bacalhau product CTAs)
Bacalhau Hover:     blue-700    #0037A6
```

### Text Colors

```
Headings:           grey-900    #000000  (or grey-800 #454545)
Body text:          grey-700    #727272
Secondary text:     grey-600    #9F9F9F
Placeholder:        grey-500    #CFCFCF
On dark bg:         #FFFFFF     (or grey-50 #FCFCFC)
```

### Backgrounds

```
Page:               grey-50     #FCFCFC  (or #FFFFFF)
Cards/Panels:       #FFFFFF     (or grey-100 #F6F6F6)
Dark mode:          grey-900    #000000
Subtle highlight:   violet-50   #FAF6FF
```

### Semantic Colors

```
Success:            green-500   #59E248  (bg: green-50 #F2FFF0)
Error:              red-500     #FF7244  (bg: red-50 #FFF3EF)
Warning:            orange-500  #FF953F  (bg: orange-50 #FFF7F0)
Info:               cyan-500    #00BBB4  (bg: cyan-50 #ECFFFE)
```

## Accessibility Quick Checks

### Minimum Contrast Ratios (WCAG AA)

```
Normal text (< 18px):     4.5:1
Large text (≥ 18px):      3:1
UI components:            3:1
```

### Safe Combinations

**On White/Light Backgrounds (#FFFFFF, grey-50)**
- ✓ grey-900, grey-800, grey-700
- ✓ violet-900, violet-800
- ✓ blue-900, blue-800
- ✓ All -900 and -800 shades

**On Dark Backgrounds (grey-900)**
- ✓ #FFFFFF, grey-50, grey-100
- ✓ violet-200, violet-100
- ✓ blue-200, blue-100
- ✓ All -200 and lighter shades

### Focus Indicators

```css
Always provide visible focus:
outline: 3px solid #883FFD (violet-600)
outline-offset: 2px
```

## Brand Rules

### Must-Follow Rules

1. **Violet** must appear in all Expanso designs
2. **Blue/Ice** must appear in all Bacalhau designs
3. When combining palettes, one must be **dominant** (>50%)
4. Never use color alone to convey information
5. Always test contrast ratios

## Presentation-Specific Adaptations

For presentations, we adapt the design system as follows:

### Dark Mode for Presentations
- Use grey-900 (#000000) as primary background
- Use white (#FFFFFF) for headings
- Use grey-50 (#FCFCFC) for body text
- Use violet-500 (#9259ED) as primary accent (more visible on dark)
- Use violet-400 for hover states

### Typography Scale (Presentation)
Following 10/20/30 rule, scale up significantly:
```
Display:    96px / 6rem      weight: 700    line-height: 1.1
H1:         72px / 4.5rem    weight: 700    line-height: 1.2
H2:         60px / 3.75rem   weight: 600    line-height: 1.3
H3:         48px / 3rem      weight: 600    line-height: 1.4
Body:       32px / 2rem      weight: 400    line-height: 1.6
Small:      28px / 1.75rem   weight: 400    line-height: 1.5
Caption:    22px / 1.375rem  weight: 400    line-height: 1.4
```

### Spacing (Presentation)
Scale up for large displays:
```
xs:    16px   1rem

md:    32px   2rem
base:  48px   3rem
lg:    64px   4rem
```

## Accessibility - WCAG AA Compliant Combinations

### On White (#FFFFFF) - 4.5:1+ contrast
✅ **Safe for body text:**
- All -900 shades (21:1 to 15:1)
- All -800 shades (12:1 to 8:1)
- grey-700 (5.74:1)
- violet-700 (6.2:1)

### On grey-50 (#FCFCFC) - Similar to white
✅ **Safe for body text:**
- All -900 shades
- All -800 shades
- grey-700 and darker

### On grey-900 / Black (#000000) - 4.5:1+ contrast
✅ **Safe for body text:**
- #FFFFFF (21:1)
- grey-50 (19.8:1)
- All -50 shades
- All -100 shades
- violet-200 and lighter (12.1:1+)
- blue-200 and lighter (14.2:1+)
- ice-300 and lighter (13.5:1+)
